# 🛡️ Danger Noodle Security
### Personal Compliance & SecOps Templates

Welcome to my personal repository. This site serves as a central hub for the compliance templates, security policies, and technical frameworks I've built. I believe security documentation should be practical, accessible, and ready for modern operations.

## 📂 Template Library

### 📋 Governance & Policy
* **[Information Security Policy (ISP)](./templates/isp-framework.md)** - A foundational security framework.
* **[Acceptable Use Policy (AUP)](./templates/aup.md)** - Defining the guardrails for corporate resource usage.
* **[Access Control Policy](./templates/access-control.md)** - Implementation of the Principle of Least Privilege.

### ⚖️ Risk & Audit
* **[Vendor Risk Assessment](./templates/vendor-assessment.md)** - A questionnaire for 3rd party security due diligence.
* **[ISO 27001:2022 Gap Analysis](./templates/iso-gap-check.xlsx)** - Checklist for organizational readiness.

### 🚨 Incident Response (IR)
* **[Ransomware Response Playbook](./templates/ir-ransomware.md)** - Technical and management response steps.
* **[Post-Incident Review (PIR)](./templates/pir-template.md)** - Template for root cause analysis and lessons learned.
